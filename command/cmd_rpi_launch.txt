RASPBERRY PI 4 - ROVER MAIN CONTROLLER
======================================
SSH: ssh curry@192.168.1.1

BUILD
-----
Build all packages:
$ cd ~/almondmatcha/ws_rpi
$ bash build.sh

Build specific package:
$ cd ~/almondmatcha/ws_rpi
$ colcon build --packages-select pkg_chassis_control

LAUNCH (Recommended)
--------------------
Launch all nodes with system launcher:
$ cd ~/almondmatcha/ws_rpi
$ source install/setup.bash
$ ros2 launch rover_launch_system rover_startup.launch.py

One-liner:
$ cd /home/curry/almondmatcha/ws_rpi && \
  source install/setup.bash && \
  ros2 launch rover_launch_system rover_startup.launch.py

MANUAL NODES (if needed)
------------------------
Terminal 1 - GNSS Position Reader:
$ cd ~/almondmatcha/ws_rpi && source install/setup.bash
$ ros2 run pkg_gnss_navigation node_gnss_spresense

Terminal 2 - Mission Monitor:
$ cd ~/almondmatcha/ws_rpi && source install/setup.bash
$ ros2 run pkg_gnss_navigation node_gnss_mission_monitor

Terminal 3 - Chassis Controller:
$ cd ~/almondmatcha/ws_rpi && source install/setup.bash
$ ros2 run pkg_chassis_control node_chassis_controller

Terminal 5 - Chassis Sensors (Domain 6):

$ cd ~/almondmatcha/ws_rpi && source install/setup.bash
$ ros2 run pkg_chassis_sensors node_chassis_sensors

Terminal 6 - Domain Bridge (Domain 5):

MONITORING
----------
Check active nodes:
$ ros2 node list

List all topics:
$ ros2 topic list

Monitor specific topics:
$ ros2 topic hz /tpc_gnss_spresense
$ ros2 topic echo /tpc_gnss_mission_active
