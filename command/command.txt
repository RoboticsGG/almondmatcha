BASE STATION (ws_base)
=======================
Quick Launch:
$ export ROS_DOMAIN_ID=2
$ cd ~/almondmatcha/ws_base && source install/setup.bash
$ ros2 run mission_control node_commands --ros-args \
  --params-file ./src/mission_control/config/params.yaml

See cmd_base_launch.txt for detailed options

STM32 FIRMWARE
==============
Build Chassis Controller:
$ cd ~/almondmatcha/mros2-mbed-chassis-dynamics
$ ./build.bash all NUCLEO_F767ZI chassis_controller
Flash build/mros2-mbed.bin via USB mass storage

Build Sensors Node:
$ cd ~/almondmatcha/mros2-mbed-sensors-gnss
$ ./build.bash all NUCLEO_F767ZI sensors_node
Flash build/mros2-mbed.bin via USB mass storage

DATA LOGGING
============
CSV Files Location: ~/almondmatcha/runs/logs/

Generated Files:
- chassis_imu_TIMESTAMP.csv      (IMU acceleration and rotation)
- chassis_sensors_TIMESTAMP.csv  (Encoder and power data)
- gnss_spresense_TIMESTAMP.csv   (GPS position records)
- lane_pub_log.csv               (Jetson lane detection)
- rover_ctl_log_ver_*.csv        (Jetson steering control)
