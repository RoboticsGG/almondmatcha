JETSON ORIN NANO - VISION NAVIGATION
=====================================
SSH: ssh yupi@192.168.1.5

BUILD
-----
Build vision navigation package:
$ cd ~/almondmatcha/ws_jetson
$ colcon build --packages-select vision_navigation
$ source install/setup.bash

LAUNCH (Recommended)
--------------------
Launch all vision nodes with launcher:
$ cd ~/almondmatcha/ws_jetson
$ source install/setup.bash
$ ros2 launch vision_navigation vision_navigation.launch.py

LAUNCH WITH OPTIONS
--------------------
Launch with camera preview and visualization:
$ ros2 launch vision_navigation vision_navigation.launch.py \
  camera_preview:=true lane_visualization:=true

Launch with video file (testing):
$ ros2 launch vision_navigation vision_navigation.launch.py \
  video_path:="/home/yupi/almondmatcha/ws_jetson/001.mp4"

Launch with custom tuning parameters:
$ ros2 launch vision_navigation vision_navigation.launch.py \
  k_p:=4.5 k_i:=0.1 k_d:=0.15 fps:=15

MANUAL NODES (if needed)
------------------------
Terminal 1 - Camera Stream Node:
$ cd ~/almondmatcha/ws_jetson && source install/setup.bash
$ ros2 run vision_navigation camera_stream

Terminal 2 - Lane Detection Node (wait 2s):
$ cd ~/almondmatcha/ws_jetson && source install/setup.bash
$ ros2 run vision_navigation lane_detection

Terminal 3 - Steering Control Node (wait 3s):
$ cd ~/almondmatcha/ws_jetson && source install/setup.bash
$ ros2 run vision_navigation steering_control

MONITORING
----------
Check camera stream:
$ ros2 topic hz /tpc_rover_d415_rgb

Monitor lane detection:
$ ros2 topic echo /tpc_rover_nav_lane

Monitor steering commands:
$ ros2 topic echo /tpc_rover_fmctl
$ ros2 topic hz /tpc_rover_fmctl

GPU OPTIMIZATION
-----------------
Check GPU usage:
$ nvidia-smi

Set CPU affinity for ROS2:
$ export ROS_DOMAIN_ID=2
$ taskset -c 0-3 ros2 launch vision_navigation vision_navigation.launch.py
